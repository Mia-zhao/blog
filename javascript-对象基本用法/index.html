JavaScript 对象基本用法

Mia Zhao Aug 8, 2020 10:05PM

这篇博文主要为了个人学习目的。

最新 ECMAScript 定义了9种类型:

  • 包括6种原始数据类型
    • undefined
    • Boolean
    • Number
    • String
    • BigInt
    • Symbol
  • null
  • Object
  • Function 并非数据结构, 属于Object, 但是有便利的 typeof instance === "function"

注意MDN中英文翻译不太一致:

  • Data Structures
  • 数据结构

此篇总结对象 (Object) 的基本用法, 包括对象的声明, 删除、查看、修改或增加对象属性。

声明对象

以下是两种声明对象的方式:

let object1 = {};

let object2 = new Object();

对象属性

一个对象可以有多个属性, 属性名和属性值构成的一个键值对依附于所属对象。对象的属性可以被删除、查看、修改或增加。

删除

对于已有对象object

let object = {'name': 'myName', 'value': 100};

删除属性value有以下方式 (注意区别):

object.value = undefined; // object['value'] = undefined;
// result: {name: "myName", value: undefined}

delete object.value; // delete object['value'];
// result: {name: "myName"}

查看

用Object.keys(object)可以查看对象object所有属性名, Object.values(object)查看对象object所有属性值, console.dir(object)查看object自身及共有属性。用object.hasOwnProperty('property')可以判断属性是否属于对象本身 (i.e., 不是共有属性)。

获得对象的某一个属性可以用如下方式 (还是以上面的object为例):

console.log(object.name);
console.log(object['name']);
// 注意与 object[name] 的区别

修改或增加

可以采用直接赋值的方法修改或增加对象属性, 以上面的object为例:

object.name = 'myNewName'; // object['name'] = 'myNewName';
object.comment = 'comment'; // object['comment'] = 'comment';

Object.assign可以批量赋值:

Object.assign(object, {'name': 'myNewName', 'comment': 'comment'});

若要修改对象共有属性, 可以用Object.create。

'name' in object VS object.hasOwnProperty('name')

'name' in object判断对象是否含有属性, object.hasOwnProperty('name')判断属性是否是对象自己的属性（不是共有属性）。

let object = {'name': 'hi'};
'name' in object;                  // true
object.hasOwnProperty('name');     // true
'toString' in object;              // true
object.hasOwnProperty('toString'); // false
