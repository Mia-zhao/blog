HTML常用标签

Mia Zhao Aug 3, 2020 9:00PM

这篇博文主要为了个人学习目的。此篇总结了HTML常用标签:

  1. a
  2. img
  3. table
  4. form

注意：应用网址浏览方式打开html文件，例如：利用工具http-server,parcel,etc.

1. <a>标签

作用

又称anchor (锚)，用于创建超链接。按功能分，可以通向：

  • 同页面的其他元素
  • 其他网页
  • 文件
  • JavaScript
  • 电子邮件/电话

以上功能通过元素的href属性达成，例如：

<!-- example 1 链接通向同页面的id="hey"的p元素 -->
<p id="hey">
<a href="#hey">example 1</a>

<!-- example 2 链接通向无协议的google.com, https://google.com或者http://google.com也是合法值-->
<a href="//google.com">example 2</a>

<!-- example 3 链接通向文件路径, "subdir1/subdir2"等效，也可以是"index.html", "./index.html"等效 -->
<a href="/subdir1/subdir2">example 3</a>

<!-- example 4 链接运行javascript代码<code>部分，如果是空则什么都不做 (不需要刷新的时候用) -->
<a href="javascript:<code>;">example 4</a>

<!-- example 5 链接发送邮件，"tel:110"则拨电话 -->
<a href="mailto:abc@example.com">example 5</a>

target属性

默认值是_self, 表明链接在<a>所在窗口打开。其他值包括：

  • _blank: 在新窗口打开，在新tab或是window打开取决于浏览器设置
  • _parent: 在<a>的父级元素环境打开，若无父级元素，效果等同_self
  • _top: 在<a>的最高祖先级元素环境打开，若无，则等同_self
  • "some_value": 任意String。若"some_value"窗口已存在，则在其窗口打开；否则新建一个"some_value"窗口并在此打开链接

download属性

指示浏览器下载，可以有值或者没有值：

  • 有值：作为下载文件名
  • 无值：浏览器提出文件名

rel属性

代表链接对象的关系，用空格分隔的Link Type表示

2. <img>标签

关键词：get request, alt, src, 响应式

<img>的功能是将图片嵌入在文本里。以下是重要属性：

  • src: 必须有的，提供图片路径
  • alt: 提供图片文本描述，不是强制属性但是有用且建议。若图片加载失败，此属性能提供具体图片信息
  • height/width: 提供任意一个，另一个相适应

重要事件：

  • onload: 加载成功事件
  • onerror: 加载失败事件，在此加error handling

重要样式：

  • 响应式(电脑，移动设备屏等)用 max-width: 100% 以下是用法：
<!-- 用法一：style  -->
<style>
  img {
      max-width: 100%;
  }
</style>

<!-- 用法二：sizes  -->
<img src="/path/to/image" alt="description" sizes="(max-width: 100%)" >

3. <table>标签

table结构：

  • <thead>/<tbody>/<tfoot>
    • <tr> (row)
      • <th>
      • <td>

table样式：

  • table-layout
    • "auto"
    • "fixed"
  • border-collapse
  • border-spacing

4. <form>标签

关键词：get/post, action, onsubmit, input

<form>表示用来提交信息的交互式控制组件。重要属性包括：

  • action: 处理提交的URL (后端)
  • method: (不区分大小写)
    • "POST": 数据通过request body发送
    • "GET": 数据附加在actionURL中
    • "DIALOG": 在<dialog>中提交表示关闭对话框
  • autocomplete ("on"/"off")
  • target: 同<a>的target属性

重要事件：

  • onsubmit

注意:

  • <form>包含input/button
  • <form>里必须有一个type "submit", 如果type没有标明，默认为"submit"
  • input (type="submit") vs button (type="submit")
    • button可以用标签包裹内容, input不可以

一个<form>例子：

<!-- 自动补全功能只有input name存在时才有用 -->
<form action="" method="POST" autocomplete="on">
  <input name="username" type="text"/>
  <input type="submit" />
</form>

input type有：

  • text, color, password, radio, checkbox, file, hidden
  • radio, checkbox 用同一name group

5. 其他感想

元素互相之间可能有关联，有reference, hierarchy等层级关系；属性选项也可能相似，要养成经常总结的习惯，并学会选取最合适的来应用。
