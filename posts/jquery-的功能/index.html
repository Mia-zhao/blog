jQuery 的功能

Mia Zhao Aug 12, 2020 3:06PM

这篇博文主要为了个人学习目的。

jQuery 是一个功能丰富的JavaScript 工具库，提供操作HTML、事件处理等API。以下简述jQuery 操作元素的功能。

1 获取元素

jQuery 主要用法在于返回操作元素的API，然后通过API 操作来实现操控元素的目的。jQuery 库通过window 对象的属性jQuery给开发者以使用jQuery 的函数和属性等，$是jQuery的别名。例如，

$( document )     // 获取整个文档
$( "*" )          // 获取所有元素
$( "#elementID" ) // 获取ID为'elementID'的元素

jQuery 在获取元素时接受CSS选择器作为参数以及jQuery 特有的选择器。所有合法的jQuery 选择器在jQuery 选择器文档中提到。

在使用$获取到某些元素后还可以用继续用find获取其子元素。例如，

$( "#elementID" ).find( "div" )  // 在获取的elementID元素的子元素中获取div元素

值得注意的是find不仅可以接受选择器为参数还接受元素或jQuery对象为参数，例如，

const allListElements = $( "li" )
$( "div" ).find( allListElements )

2 链式操作

jQuery 的函数返回jQuery 对象使得链式操作成为可能。链式操作使代码更简洁，在通过选择器获取元素时只需获取一次就能实现多重操作。例如，

$( "div" ).addClass( "new-class" ).css( "color", "red" )
// 给所有div元素加一个类"new-class"并把样式颜色设置为"red"

这样的操作可以多重进行下去，代码实现的功能也一目了然。如果要获取子元素或者返回子元素上一层，可以用find和end。

3 创建元素

用append可以实现创建元素的目的，将特定内容插入到匹配元素的最后作为最后一个子元素。用prepend插入作为第一个子元素。appendTo实现与append相同的功能，只是语法不同，a.append(b)与b.appendTo(a)功能等价。

append接受字符串、元素、jQuery 对象等作为参数。

$( "body" ).append( "<p></p>" )
$( "p" ).append( document.createTextNode("Hi") )
$( "p" ).append( $( "strong" ) )

4 移动元素

利用元素插入的函数实现。例如，insertAfter在目标元素后插入匹配元素，考虑如下HTML:

<div class="container">
  <h2>Greetings</h2>
  <div class="inner">Hello</div>
  <div class="inner">Goodbye</div>
</div>

在页面上选择h2并插入在div后达到移动元素的目的。
$( "h2" ).insertAfter($( ".container" ))结果为

<div class="container">
  <div class="inner">Hello</div>
  <div class="inner">Goodbye</div>
</div>
<h2>Greetings</h2>

其他类似的函数还有after, insertBefore, before, appendTo, append, prependTo, prepend。需要注意的是这些函数的语法，特别是内容和目标元素的顺序以及函数返回的是内容对象还是目标元素对象。

5 修改元素属性

jQuery 函数可以获取元素属性或设置元素属性。常见的有:

.addClass()     // 为每个匹配元素添加类
.attr()         // 获取或设置第一个匹配元素的attribute
.html()         // 获取或设置每一个匹配元素的html内容
.prop()         // 获取或设置第一个匹配元素的property
.removeAttr()   // 移除每一个匹配元素的一个或多个attribute
.removeClass()  // 移除每一个匹配元素的一个或多个样式类
.removeProp()   // 移除每一个匹配元素的一个或多个property
.toggleClass()  // 添加或删除每一个匹配元素的样式类
.val()          // 获取或设置第一个匹配元素的值

注意 attribute 和 property 区别。attribute 是 HTML 中键值对，property 是对应的DOM属性。
