2 重要概念

2.1 文档流 normal flow

在默认布局下，文档流可以是左到右或是上到下。例如，span元素是默认合并的(inline), div元素是默认占一行不合并的(block): 除此以外，inline元素不能指定width/height, 元素的高度由line-height间接确定。block元素可以指定width/height,元素高度由内部文档流元素决定。

注：文档流元素是指处于文档流的元素，position: [absolute|fixed] 以及 float会使元素脱离文档流。

除了inline和block, 还可以通过display: inline-block;设置元素为inline-block。inline-block兼顾inline和block两者的特点，宽度流动与inline一致，高度流动与block一致，可以通过width/height设置宽高。

注意：

  1. block默认width的值是auto而非100%
  2. 不要用设置width为100%
  3. inline元素不能包含block元素

2.2 盒模型 box model

CSS盒模型分为:

  • 内容盒模型(content box)
  • 边框盒模型(border box)

区别在于内容盒模型宽度高度设置只包含内容(content)，边框盒模型宽度高度设置包含内容(content), 内边距(padding), 边框(border)。 CSS里盒模型用box-sizing设置，默认用的是内容盒模型，建议用border-box。
box-sizing: content-box; /*内容盒模型*/
box-sizing: border-box; /*边框盒模型*/

3 布局

CSS布局可以将页面分为上中下或左中右等结构，分为固定宽度布局，不固定宽度布局，响应式布局。布局通过float, flex, grid完成。

3.1 float布局

步骤：

  1. 子元素加float 和 width
  2. 父元素加.clearfix
.clearfix:after {
  content: '';
  display: block;
  clear: both;
}

3.2 flex布局

可以通过flex 青蛙游戏熟悉flex属性用法。

flex设置分为container和items设置，container和items举例如下：

<div class="menu">
  <div class="logo">
  </div>
  <ul>
    <li>menu 1</li>
    <li>menu 2</li>
    <li>menu 3</li>
  </ul>
</div>

以上例子需要CCS reset:

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
ul, ol {
  list-style: none;
}

常用container元素属性有：

  • display: [flex|inline-flex]
  • flex-direction: [row|row-reverse|column|column-reverse] 定义主轴方向
  • flex-wrap: [nowrap|wrap|wrap-reverse] 改变折行
  • justify-content: [flex-start|flex-end|center|space-between|space-around] 设置主轴对齐方式
  • align-items: [flex-start|flex-end|center|stretch] 设置次轴对齐方式

常用items元素属性有：

  • order 元素按照order值的大小进行排序
  • flex-grow 设置变宽系数
  • flex-shrink 设置变窄系数，0防止缩短
  • flex-basis 基准宽度，默认auto
  • flex: flex-grow flex-shrink flex-basis 属性缩写
  • align-self 单独设置align-items

3.3 grid布局

flex是一维布局方式，grid是二维布局方式。可以通过grid 花园游戏熟悉grid用法。

grid也分为container和items, 与flex类似。通过display: [grid|inline-grid]设置container, container可以被划分：

.container {
    grid-template-columns: 40px auto 40px;
    grid-template-rows: 25% 100px auto; 
}

以上将container分为三列三行, 每一个item可以通过划分的结构设置范围：

.item1 {
    grid-column-start: 1;
    grid-column-end: 3;
    grid-row-start: 1;
    grid-row-end: 2;
}
